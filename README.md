# 🏭 DM管理系统 V4 — 日常管理 (Daily Management System)

生产数据管理、SQDIP看板、异常LOSS管控、PSP问题闭环、设备点检等全功能工厂管理平台。

## 功能模块

- **底账录入** — 每日产出/工时/效率数据录入
- **SQDIP看板** — 安全、品质、交付、人员、效率全维度监控
- **异常LOSS管控** — 异常损失实时追踪，关联PSP改善
- **PSP问题闭环** — 问题原因分析、PDCA改善、闭环管理
- **经营全景报告** — 月度/周度生产总结报告，AI智能复盘
- **系统运作** — 运行率、人员、设备等关键KPI监控
- **设备点检** — 日/周/月设备巡检与状态追踪
- **目标管理** — 少人化、效率、交付等目标追踪
- **标准工时/人员推演** — 产能与人员匹配性推演
- **改善提案** — 改善项目跟踪管理

## 技术栈

- 原生 HTML/CSS/JavaScript
- Firebase Realtime Database (数据同步)
- Chart.js (图表可视化)
- XLSX (Excel导出)
- html2canvas + jsPDF (报告打印/导出)

## 使用说明

打开 `combined.html` 即可运行（所有资源内联）。
开发版本使用 `index.html` + `css/styles.css` + `js/` 外部加载。

> 时区: Asia/Bangkok (GMT+7)
> 语言: 中文/English/ไทย (多语言支持)
